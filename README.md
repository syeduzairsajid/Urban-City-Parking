
# Urban City Parking System

 
This repository is the original group project repository created for the assessment.
It contains the base implementation of the Urban City Parking System.

The project manages parking sessions, vehicle entry and exit, and pricing logic
using a structured and modular design.

## Group Project Information
This repository serves as the main project repository. Other group members have
forked this repository to implement their individual contributions as required
